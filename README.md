```text

        ↓
```

* routes

## 🚀 Local Setup

### 1) Clone repository

```bash
git clone https://github.com/your-user/documind-ai-enterprise.git
cd documind-ai-enterprise
```

### 2) Backend

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3) Frontend

```bash
cd frontend
npm install
npm run dev
```
